## Renef

Renef is a dynamic instrumentation toolkit for Android ARM64. It enables runtime code analysis and manipulation for security research, reverse engineering, and CTF challenges.

### Key Features

- ARM64 function hooking (PLT/GOT and inline trampoline)
- Lua 5.4 scripting engine with Frida-like API
- Process injection without ptrace
- Memory scanning, reading, writing, and patching
- Java method hooking via JNI
- Interactive TUI memory scanner
